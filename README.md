# major-manager-server
 


## Project setup
```
npm install
```

* create a file called db.config.js that exports a json object with the following database information: 
```
module.exports = {
  HOST: 'localhost',
  DB: <database-name>,
  USER: <username>,
  PASSWORD: <password>,
  PORT: <port default is set to 3306>
};

```

### Run
```
npm start
```
### Test
```
npm run test
```
### Linter
```
npm run lint
```
